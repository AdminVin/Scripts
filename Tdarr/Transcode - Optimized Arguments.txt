[>] Configuration - Set Tdarr to CLI
- Transcode Options - Set to FFMPEG CLI
 - Navigate to [Library > Select Library > Transcode Options > Check off "Basic Video" > Check 'FFMPEG' > Change 'CLI arguments']


[>] Arguments
- Resolution
 - Downscale all content to 720P
  - Add: -vf "scale='if(gt(ih,720),-2,iw)':720"
 - Downscall all content to 1080P
  - Add: -vf "scale='if(gt(ih,1080),-2,iw)':1080"

* Audio
- To transcode to the AC3 codec use '-c:a ac3 -b:a 640k -metadata:s:a language=eng'
- To transcode to the AAC codec use '-c:a aac -b:a 320k -metadata:s:a language=eng'
- To skip audio transcoding, and copy existing audio use '-c:a copy'

* Subtitles
- Add '-sn' to remove subtitles


[>] Options
* Filtering
- Resolution
 - Navigate to [Library > Select Library > Transcode Options > Filters]
 - Filters at transcode queue level: 480p,570p,576p
 - Any files meeting the resolutions will be skipped from transcoded.

- Size 
 - Adjust slider on "Video Size Range" to cutoff of choice (Recommended: TV Shows 1GB / Movies 10GB)
 - Any files smaller than the cutoff MB will be skipped from being transcoded.

- Disable double encoding of any HEVC format files
 - Under "Transcode Options" 
 - Select "dont", and type in "hevc" and then enter > Check off the box for "hevc" so that any files in HEVC format will be skipped.

[>] Library Configurations
  [>] TV Shows
  * Resolution to 720p / Convert to HEVC

[Mac Mini M1 (2024/04/15)]
<io> -vcodec hevc_videotoolbox -quality high_quality -profile:v main -preset veryslow -b:v 3000k -maxrate 5000k -bufsize 10000k -vf "scale='if(gt(ih,720),-2,iw)':720" -c:a ac3 -b:a 640k -metadata:s:a language=eng -metadata:s:a language=eng -avoid_negative_ts make_zero -max_muxing_queue_size 4096 -ignore_chapters 1 -movflags +faststart -threads 4
